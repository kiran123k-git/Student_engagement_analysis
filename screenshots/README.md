## 🔗 Adding Screenshots

### Steps:
1. Take screenshot of each page
2. Save as PNG format (high resolution: 1920×1080)
3. Name: `NN-page-name.png` (e.g., `01-dashboard-home.png`)
4. Place in this directory
5. Update links above

### Screenshot Tools:
- macOS: Command + Shift + 4
- Windows: Snipping Tool
- Online: Screenshot.guru, CloudShot

### Best Practices:
- ✅ Include UI elements and data
- ✅ Highlight important features
- ✅ Use consistent styling
- ✅ Avoid sensitive student data
- ✅ Use demo/sample data
