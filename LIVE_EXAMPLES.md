## Data Integrity Guarantees

### ✅ ONLY REAL DATA USED

Every student mentioned is **actually in the dataset**.
Every metric shown is **calculated from real student records**.
Every percentage is **based on actual data**, not estimates.

### ✅ NO FABRICATED STUDENTS

System will **never invent** students like:
- "Student X with engagement 95%"
- "A hypothetical high performer"
- "An example struggling student"

### ✅ NO GENERIC STATEMENTS

System will **never provide**:
- "Students should work harder"
- "Encourage study groups"
- "Time management tips"
- "Generic counseling advice"

### ✅ MISSING DATA HANDLED GRACEFULLY

If a field is missing:
- System uses available fields
- Clearly notes what data is missing
- Still provides useful insights
- Never forces empty data into response

---

## Error Handling

### If Query Cannot Be Answered:

**System says:**
```
"Insufficient data in retrieved records to answer this query."
```

**Examples:**
- Student name not in database → "Student 'John Doe' not found in database"
- Required metric missing → "Cannot calculate engagement without attendance data"
- No students in category → "No high-performing students identified"
